This package (`electricalpython`) includes both `eepower` and `eesignal` which each contain several sub-packages that are discretely broken up in a logical fashion. The package-breakdown is described below.

1. EEPOWER
    - CAPACITOR
    - PERUNIT
2. EESIGNAL
